# Engineering Principle #005

## Principle

Every file exists for a reason.

---

## Meaning

Nothing is added to the repository without a clear purpose.

---

## Why It Matters

A clean repository is easier to maintain, navigate, and understand.

---

## Example

The Knowledge Base, Roadmap, Manifesto, and Engineering Decisions each answer a different question.

---

## Key Takeaway

If a file has no purpose, it should not exist.
