# Engineering Principles

## Purpose

Engineering Principles define how this academy thinks.

They are timeless guidelines discovered through experience rather than copied from books.

These principles apply to every project, every lesson, and every future language repository built using this system.

---

## What You'll Find Here

Each document contains one engineering principle that has been earned through hands-on experience while building this academy.
