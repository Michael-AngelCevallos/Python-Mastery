# Engineering Decision #005

## Title

Standard Project Format

---

## Status

Accepted

---

## Context

Projects should all follow the same layout so every repository is immediately understandable.

---

## Decision

Every project includes a consistent structure including:

- main.py
- README.md
- Notes
- Supporting files
- Assets (when needed)

---

## Why

A standard project layout improves maintainability and readability.

---

## Alternatives Considered

- Custom structure for every project.

---

## Consequences

### Positive

✔ Familiar layout

✔ Easier onboarding

✔ Better documentation

### Negative

• Some very small projects may contain extra files.
